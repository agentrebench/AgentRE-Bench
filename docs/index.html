AgentRE-Bench

Long-horizon reverse engineering tasks for LLM agents, scored deterministically.

Give an LLM agent a compiled binary and a disassembler. No source code. No hints. Measure how well it identifies C2 infrastructure, encryption, anti-analysis, and protocols. 23 levels across Linux (ELF x86-64) and Windows (PE32+ x86-64).

Why This Benchmark

What makes AgentRE-Bench different

Simple Q&A benchmarks don't differentiate frontier models on real-world reasoning. AgentRE-Bench tests chains of 10–25 tool calls where each output informs the next decision.

✦

Synthetic

All 23 binaries (13 ELF + 10 PE32+) compiled from C/C++ sources. Known ground truths, no licensing issues.

≡

Deterministic

Every answer scored against fixed ground truth with weighted fields and Jaccard overlap. No LLM-as-judge.

⚙

Agentic

Agents plan tool use, interpret raw output (hex dumps, disassembly), synthesize within a 25-call budget.

⇆

Long-Horizon

Hardest levels: entropy analysis, key extraction, decryption, identifying 18+ anti-* techniques.

Scoring Model

Deterministic, weighted scoring

Each of the 21 standard levels is scored across 5 weighted fields. Levels 13 and 23 are bonus tasks with a 10-field rubric. Maximum total score is 3.0.

Standard Levels (1–12, 14–22)

decoded_c240%
techniques30%
file_type10%
encoded_strings10%
c2_protocol10%
−0.05 per fabricated technique

Bonus Levels (13 & 23)

decoded_c215%
encryption_algorithm10%
encryption_key15%
encryption_key_storage5%
techniques15%
decoded_strings15%
anti_analysis10%
file_type / strings / protocol10%
−0.03 per fabricated technique

Aggregate Score

Main = average(level_1 … level_12, level_14 … level_22) → 0.0 – 1.0
Bonus = level_13_score + level_23_score     → 0.0 – 2.0
Total = Main + Bonus           → 0.0 – 3.0

Per-model findings

Gemini 3.1 Flash Lite non-thinking

Bench leader. Wins or ties on 8 of 13 levels including L7 DNS Tunnel (only model above 0.33), L11 Fork Bomb (0.80), L12 JIT Shellcode (0.76). Lowest hallucination rate (1.92/task). 525s total, 30.5M tokens.

DeepSeek V4 Pro thinking

Only model to crack the bonus meaningfully (L13 = 0.271). Strong calibration (2.69/task). Lost L3,4,5 to timeouts; would likely lead Main if those held. Slowest: 920s avg, 3338s total.

Claude Opus 4.7 adaptive thinking

Most reliable: 13/13 valid answers, 0 errors. Strongest on L8 Process Hollow (0.54). Adaptive thinking surfaces broader candidates — 5.62/task, ~3.6pt penalty.

Kimi K2.6 reasoning

Best on L1 (0.875) and L2 (0.83) by clear margins. Second-lowest hallucination (2.08/task). Lost 4 tasks to mid-conversation HTTP 400s.

DeepSeek V4 Flash non-thinking

Strong on L1 (0.72), L5 (0.69), L11 (0.71). 13/13 answers, 636s total. Outperforms two thinking models on Main. Submitted empty for L13.

GPT-5.5 reasoning

Highest hallucination (6.31/task). L13 burned 22 min, 72,493 reasoning tokens for 0 score. Only model with reasoning-token accounting.

Findings

Where models break

Across 7 frontier models and 13 RE tasks, the failures cluster into recognizable patterns. None are about model size or thinking budget — they're about specific cognitive moves current LLMs cannot reliably execute on a binary.

⚠

Nested misdirection

L10 AES Encrypted Shell — max 0.15. Binary advertises AES, actually 16-byte XOR. Every model trusts the labels. Nobody traces control flow to disprove the lie.

≡

Symbolic execution at scale

L13 — metadata 1.0, execution 0.0. Models identify "RC4 with key X" but can't mentally run RC4 across 4 KB. Recognition layer solved; in-head computation is the wall.

↻

Polymorphism confusion

L4 Polymorphic Shell — max 0.05. Random NOP sled, but C2 IP/port are static. Every model reads "polymorphic" and concludes "unknowable."

⇔

Stateful protocol RE

L7 DNS Tunnel — 5 of 6 models below 0.10. TXT-record commands across multiple beacon/response pairs. Reconstructing framing requires sustained control-flow reasoning.

✎

Over-claiming

Hallucination rank ≈ score rank. GPT-5.5: 6.31/task (~4.1pt penalty). Opus: 5.62/task (~3.6pt). Flash Lite (1.92) and Kimi (2.08) lead by not claiming what they can't defend.

↻

Reasoning runaway

GPT-5.5 L13: 22 min, 72K tokens, 0 score. Reasoning trace attempted symbolic decryption, got lost, never converged. Expensive zeros.

Getting Started

Up and running in 5 steps

Zero Python dependencies. All LLM provider calls use Python's built-in urllib. Just bring an API key.

1

Clone & configure

Clone the repo and add at least one provider API key.

git clone https://github.com/agentrebench/AgentRE-Bench.git cd AgentRE-Bench cp .env.example .env # add your API key(s)
2

Build Linux binaries

Compile the 13 ELF64 challenge binaries. Uses local gcc on Linux, Docker on macOS.

chmod +x build_binaries.sh ./build_binaries.sh
3

Build Windows binaries

Cross-compile 10 PE32+ challenge binaries. MinGW or Docker fallback.

chmod +x build_windows_binaries.sh ./build_windows_binaries.sh
4

Build tools image

Build the Docker image for sandboxed tool execution.

docker build --platform linux/amd64 \ -t agentre-bench-tools:latest \ -f Dockerfile.tools .
5

Run the benchmark

Run a single task or the full 23-level suite.

python run_benchmark.py --task level1_TCPServer -v python run_benchmark.py --all --provider anthropic
